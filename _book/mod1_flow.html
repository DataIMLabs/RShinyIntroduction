4  Flow

Flow-control in R is een essentieel concept dat je helpt om beslissingen te nemen in je code, herhaalde taken uit te voeren, en door datastructuren te itereren. We zullen kijken naar de meest voorkomende flow-control structuren in R: conditionele statements (if, else if, else), loops (for, while), en de repeat loop. We zullen ook kort de break en next statements aanraken.

4.1 Voorwaardelijke uitvoer

Conditionele statements laten je toe om te beslissen welke code uitgevoerd moet worden afhankelijk van bepaalde voorwaarden.

4.1.1 if / else

  • R
x <- 5

if (x > 0) {
  print("x is positief")
} else if (x < 0) {
  print("x is negatief")
} else {
  print("x is nul")
}

4.1.2 switch

In R kan een switch-statement gebruikt worden om één van vele codeblokken te selecteren die uitgevoerd moeten worden. Een belangrijk kenmerk van het switch-statement is dat het direct het resultaat van de overeenkomende case teruggeeft. Dit betekent dat je het resultaat van een switch-statement direct kunt toewijzen aan een variabele.

  • R
# Definieer de variabele
dag <- 4

# Gebruik switch om de waarde van de variabele te vergelijken met verschillende gevallen
resultaat <- switch(dag,
  "1" = "Maandag",
  "2" = "Dinsdag",
  "3" = "Woensdag",
  "4" = "Donderdag",
  "5" = "Vrijdag",
  "6" = "Zaterdag",
  "7" = "Zondag",
  "Onbekende dag"
)

# Print het resultaat
print(resultaat)

4.2 Loops

Loops maken het mogelijk om code meerdere keren uit te voeren.

4.2.1 For Loops

Een for-loop werkt als volgt:

  • R
# Initialiseer een vector
vect <- 1:5

for (i in vect) {
  print(i^2)  # Kwadrateer elk element en geef de output weer 
}

4.2.2 While-loop

  • R
  • R2
count <- 1

while(count <= 5) {
    print(count)
    count <- count + 1
}

test

4.2.3 Repeat Loops

De repeat loop voert een blok code uit tot het onderbroken wordt door een break statement.

  • R
count <- 1

repeat {
    print(count)
    if(count >= 5) {
        break  # Verlaat de loop als count groter of gelijk aan 5 is
    }
    count <- count + 1
}
